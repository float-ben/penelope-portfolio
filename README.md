# Penelope Cai Portfolio

Penelope's portfolio website!

## Files

- `index.html` - page structure and resume content
- `styles.css` - editorial visual system and responsive layout
- `script.js` - scroll reveals, active navigation, and subtle pointer interaction
- `assets/Penelope_Cai_resume.pdf` - downloadable resume
- `assets/pen_headshot-web.jpg` - optimized headshot for the site

## GitHub Pages

For a personal site, create a repository named `your-github-username.github.io`, upload these files to the root of the repository, and GitHub Pages will serve the site from `https://your-github-username.github.io`.

For a project site, upload these files to any repository, then enable GitHub Pages in repository settings using the `main` branch and `/root` folder.
